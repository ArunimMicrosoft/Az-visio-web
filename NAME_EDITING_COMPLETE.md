## 🎯 How to Edit Names

### Icons (Azure Services):
1. **Double-click** the name below the icon
2. Type your new name
3. Press **Enter** to save (or **Escape** to cancel)

### Boundaries:
1. **Click** the boundary label in the header
2. Type your new name
3. Press **Enter** to save (or **Escape** to cancel)

---
