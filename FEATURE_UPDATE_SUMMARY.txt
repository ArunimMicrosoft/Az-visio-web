✨ NEW FEATURE #1: CONNECTION VALIDATION WITH LED INDICATORS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔗 What is it?
   The app now validates Azure service connections and shows visual 
   feedback using colored LED indicators at the connection midpoint.

🎨 LED Color Meanings:
   🟢 GREEN    = Valid Azure connection (recommended)
   🟡 YELLOW   = Uncommon connection (verify your design)
   🔴 RED      = Invalid connection (not recommended)

💡 How it works:
   1. Connect two services using SHIFT + Click
   2. LED automatically appears at the midpoint
   3. LED pulses with animation
   4. Hover over LED to see validation message
   5. Connection line and arrow match LED color

📋 Validation Examples:
   ✅ VM → Storage Account          = 🟢 Valid
   ✅ App Service → SQL Database    = 🟢 Valid
   ⚠️  VM → Cosmos DB                = 🟡 Warning (uncommon)
   ❌ Invalid patterns              = 🔴 Red (customizable)
