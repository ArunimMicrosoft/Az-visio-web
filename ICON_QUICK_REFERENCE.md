## 💡 Usage Tips

### Finding Icons
1. **By Category**: Browse categories in the toolbar
2. **By Name**: Look for service name in the category
3. **By Use Case**: 
   - Web apps → Compute (App Services, Function Apps)
   - Data → Databases or Storage
   - Networking → Networking category
   - Security → Security or Identity

### Common Architecture Patterns

#### **3-Tier Web Application**
```
Internet → Application Gateway
         → App Service
         → SQL Database
         → Storage Account
```

#### **Microservices**
```
Internet → API Management
         → AKS (Kubernetes)
         → Service Bus
         → Cosmos DB
         → Container Registry
```

#### **Data Analytics Platform**
```
Data Sources → Event Hubs
             → Stream Analytics
             → Data Lake Storage
             → Synapse Analytics
             → Power BI
```

#### **Hybrid Cloud**
```
On-Premises → ExpressRoute
            → Virtual Network
            → Azure Arc
            → Azure Monitor
```
