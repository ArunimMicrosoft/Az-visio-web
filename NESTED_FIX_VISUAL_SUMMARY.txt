┌───────────────────────────────────────────────────────────────┐
│ ROOT CAUSE                                                    │
├───────────────────────────────────────────────────────────────┤
│                                                               │
│  🔍 Z-Index Stacking Context Issue                           │
│                                                               │
│     Parent Subscription (z-index: auto)                       │
│       └─ Child Resource Group (z-index: auto)                │
│            └─ Label (z-index: 100)                           │
│                                                               │
│     Problem: Child z-index only relative to parent!          │
│     Parent's stacking context BLOCKED child label clicks!    │
│                                                               │
│  🔍 Event Propagation Issue                                  │
│                                                               │
│     Click → Parent captures → Child never receives!          │
│                                                               │
└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐
│ SOLUTION (AFTER)                                              │
├───────────────────────────────────────────────────────────────┤
│                                                               │
│  ✅ DYNAMIC Z-INDEX ELEVATION                                │
│                                                               │
│     When editing starts:                                      │
│     • Boundary jumps to z-index: 999                         │
│     • Header gets z-index: 1000                              │
│     • Label gets z-index: 1001                               │
│     • Input field gets z-index: 1002 (HIGHEST!)              │
│                                                               │
│  ✅ EVENT PROPAGATION CONTROL                                │
│                                                               │
│     onClick={(e) => {                                         │
│       e.stopPropagation();  ← Blocks parent                  │
│       e.preventDefault();   ← Stops defaults                 │
│       startEditing();                                         │
│     }}                                                        │
│                                                               │
│  ✅ SMART DRAG vs EDIT DETECTION                             │
│                                                               │
│     • Click LABEL → Edit mode (no drag)                      │
│     • Click HEADER → Drag mode (no edit)                     │
│                                                               │
└───────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────┐

├───────────────────────────────────────────────────────────────┤
