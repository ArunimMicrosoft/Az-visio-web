# Azure Architecture Designer

<!-- Deployment: Updated token -->

A modern web application for designing Azure architecture diagrams, similar to Microsoft Visio. Create professional cloud architecture diagrams with drag-and-drop functionality.

## 🚀 Features

- **Drag & Drop Interface**: Easily drag Azure service icons from the toolbar onto the canvas
- **500+ Azure Service Icons**: Official Azure SVG icons organized into 8 categories (Compute, Storage, Databases, Networking, Security, Integration, Monitoring, AI + ML)
- **Category Navigation**: Browse icons by service category with intuitive tab navigation
- **Visual Connections**: Create connections between components by holding Shift and clicking on items
- **Save & Load**: Save your diagrams to localStorage and load them later
- **Export**: Export your architecture diagrams as JSON files
- **Interactive Canvas**: Move items around, delete them, and organize your architecture
- **Modern UI**: Clean and intuitive interface with Azure color scheme
- **Official Azure Icons**: Uses actual Microsoft Azure SVG icons for professional diagrams

## 🛠️ Tech Stack

- **React** - UI framework
- **Vite** - Build tool and dev server
- **CSS3** - Styling with modern flexbox/grid layouts
- **HTML5 SVG** - For drawing connections

## 🚦 Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit `http://localhost:5173`

## 📖 How to Use

1. **Add Services**: Drag Azure service icons from the left toolbar onto the canvas
2. **Move Items**: Click and drag items on the canvas to reposition them
3. **Create Connections**: Hold `Shift` key and click on two items to draw a connection line
4. **Delete Items**: Select an item (click on it) and press the `Delete` key
5. **Save/Load/Export**: Use the control buttons at the top

## 🔧 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
