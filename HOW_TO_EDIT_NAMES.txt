# ✏️ HOW TO EDIT NAMES - QUICK GUIDE

## For Azure Service Icons (VMs, Storage, Databases, etc.):

**Step 1:** Place an icon on the canvas  
**Step 2:** **DOUBLE-CLICK** the name below the icon  
**Step 3:** Type your new name  
**Step 4:** Press **ENTER** to save (or **ESC** to cancel)  

```
Before:              After:
┌─────────┐         ┌──────────────────────┐
│   💻    │         │         💻           │
│  VM 1   │  →→→    │ prod-web-server-01   │
└─────────┘         └──────────────────────┘
   ↑                          ↑
Double-click here!      New custom name!
```

---

## For Boundaries (Resource Groups, VNets, Subnets, etc.):

**Step 1:** Draw a boundary (click Boundary button, select type, drag on canvas)  
**Step 2:** **CLICK** the boundary name in the colored header  
**Step 3:** Type your new name  
**Step 4:** Press **ENTER** to save (or **ESC** to cancel)  

```
Before:
┌────────────────────────────────────┐
│ 📦 Resource Group 1    [0 services]│ ← Click here!
│                                    │
│                                    │
└────────────────────────────────────┘

After:
┌────────────────────────────────────┐
│ 📦 rg-prod-eastus      [0 services]│
│                                    │
│                                    │
└────────────────────────────────────┘
```

---

## Keyboard Shortcuts:

⌨️ **ENTER** = Save changes  
⌨️ **ESCAPE** = Cancel editing  
🖱️ **Click outside** = Auto-save
