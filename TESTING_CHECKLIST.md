## 🐛 Common Issues & Solutions

### Issue: Blank White Page
**Solution**: 
1. Check browser console for errors
2. Verify dev server is running (`npm run dev`)
3. Refresh page (Ctrl+F5)
4. Check that all imports are correct

### Issue: Icons Not Loading
**Solution**: 
1. Check `/public/icons/` folder exists
2. Verify icon paths in `azureIcons.js`
3. Check browser console for 404 errors

### Issue: PDF Export Fails
**Solution**: 
1. Check browser console for error message
2. Verify jsPDF is installed (`npm list jspdf`)
3. Make sure items array is not empty
4. Check that calculateCost returns valid data

### Issue: Cost Not Updating
**Solution**: 
1. Check that CostSummary component is rendered
2. Verify costCalculator.js has pricing for the service
3. Check browser console for errors

### Issue: Name Editing Doesn't Work
**Solution**: 
1. Make sure you're double-clicking the label (not icon)
2. Check that editingItemId state is updating
3. Verify input field appears
